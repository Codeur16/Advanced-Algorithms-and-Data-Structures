# Algorithmes de Résolution de Problèmes

Ce projet regroupe plusieurs algorithmes classiques pour la résolution de problèmes courants en informatique. Vous y trouverez des implémentations de la recherche binaire, des parcours de graphes (BFS et DFS), du problème du sac à dos (0/1 Knapsack), de la fusion des intervalles et de la somme maximale d'un sous-tableau (Kadane).

## Prérequis

Avant de commencer, assurez-vous d'avoir Python 3.x installé sur votre machine. Vous pouvez vérifier cela en exécutant la commande suivante :

## Installation
> Clonez ce dépôt ou téléchargez les fichiers sources.
```bash
git clone https://votre-lien-repository.git
cd votre-dossier
```
>  Créez un environnement virtuel pour installer les dépendances.
```bash 
python -m venv venv
source venv/bin/activate  # Sur Linux ou MacOS
venv\Scripts\activate     # Sur Windows
```
> Installez les dépendances nécessaires.
```bash
    pip install -r requirements.txt
```
## Lancer l'application
```bash
python main.py
```
> Interface interactive
```bash 
=============== Sélectionnez un algorithme =================
1. Recherche binaire
2. Parcours de graphe (BFS)
3. Parcours de graphe (DFS)
4. Problème du sac à dos
5. Fusion des intervalles
6. Somme maximale d'un sous-tableau (Kadane)
7. Quitter
```
